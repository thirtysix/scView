# scView Development Guide

## Development Environment Setup

### Prerequisites

- Docker Desktop (v20.10+) with Docker Compose v2
- 8 GB+ RAM allocated to Docker
- Optional: Python 3.11+ and Node.js 20+ for local development without Docker

### Quick Start

```bash
# Clone the repository
git clone <repository-url> scView
cd scView

# Copy environment config
cp .env.example .env

# Start all services with hot-reload
make dev
```

Services will be available at:
- **Frontend**: http://localhost:5173 (Vite dev server with HMR)
- **Backend API**: http://localhost:8000 (Uvicorn with auto-reload)
- **R Converter**: http://localhost:8001 (httpuv server)
- **API docs**: http://localhost:8000/docs (Swagger UI, auto-generated by FastAPI)

### Makefile Targets

```bash
make dev        # Start all services in development mode (hot-reload)
make build      # Build production Docker images
make up         # Start production services
make down       # Stop all services
make logs       # Tail logs from all services
make test       # Run backend tests (pytest)
make clean      # Remove containers, images, and volumes
make setup      # Initial setup: copy .env, create data dirs
```

## Backend Development

### Adding a New API Endpoint

1. Create a new file in `backend/src/scview/api/v1/`:

```python
"""My new endpoint."""

from fastapi import APIRouter, Depends, HTTPException
from scview.dependencies import get_dataset_manager
from scview.core.dataset_manager import DatasetManager

router = APIRouter()

@router.get("/datasets/{dataset_id}/my-endpoint")
async def my_endpoint(
    dataset_id: str,
    dm: DatasetManager = Depends(get_dataset_manager),
):
    adaptor = await dm.get_or_load_dataset(dataset_id)
    if adaptor is None:
        raise HTTPException(status_code=404, detail="Dataset not found.")

    # Use adaptor to access data
    return {"result": "..."}
```

2. Register the router in `backend/src/scview/api/router.py`:

```python
from scview.api.v1.my_endpoint import router as my_endpoint_router

api_router.include_router(my_endpoint_router, prefix="/v1", tags=["my-endpoint"])
```

### Returning Arrow IPC Binary

For large numerical data, use Arrow IPC instead of JSON:

```python
from fastapi.responses import Response
from scview.core.arrow_serializer import dataframe_to_arrow_ipc

@router.get("/datasets/{dataset_id}/my-data")
async def get_my_data(...):
    df = adaptor.get_some_dataframe()
    ipc_bytes = dataframe_to_arrow_ipc(df)
    return Response(content=ipc_bytes, media_type="application/vnd.apache.arrow.stream")
```

## Frontend Development

### Adding a New Panel

1. Create the panel component in `frontend/src/components/panels/`:

```tsx
// MyNewPanel.tsx
import { useQuery } from '@tanstack/react-query';
import { apiFetch } from '../../api/client';
import { useDatasetStore } from '../../stores/datasetStore';

export function MyNewPanel() {
  const datasetId = useDatasetStore((s) => s.currentDatasetId);

  const { data, isLoading, error } = useQuery({
    queryKey: ['my-data', datasetId],
    queryFn: () => apiFetch(`/datasets/${datasetId}/my-endpoint`),
    enabled: !!datasetId,
  });

  if (isLoading) return <div>Loading...</div>;
  if (error) return <div>Error: {error.message}</div>;

  return (
    <div className="p-6">
      <h2 className="text-xl font-semibold mb-4">My Panel</h2>
      {/* Panel content */}
    </div>
  );
}
```

2. Register the panel in `frontend/src/lib/constants.ts`:

```typescript
export const PANEL_IDS = [..., 'my-panel'] as const;
export const PANEL_LABELS: Record<PanelId, string> = {
  ...,
  'my-panel': 'My Panel',
};
```

3. Add it to `PanelContainer.tsx`:

```typescript
import { MyNewPanel } from '../panels/MyNewPanel';

const PANEL_COMPONENTS: Record<PanelId, React.FC> = {
  ...,
  'my-panel': MyNewPanel,
};
```

4. Add a sidebar icon in `Sidebar.tsx`.

### Working with Arrow IPC Data

For fetching and decoding binary Arrow data:

```tsx
import { useQuery } from '@tanstack/react-query';
import { apiFetchBinary } from '../../api/client';
import { decodeArrowBuffer } from '../../lib/arrow';

const { data } = useQuery({
  queryKey: ['embedding', datasetId, embeddingName],
  queryFn: async () => {
    const buffer = await apiFetchBinary(`/datasets/${id}/embeddings/${name}`);
    return decodeArrowBuffer(buffer);
  },
});
```

For performance-critical paths (200k+ rows), use the Web Worker:

```tsx
// See useEmbedding.ts for the full pattern
const worker = new Worker(
  new URL('../../workers/arrowDecoder.worker.ts', import.meta.url),
  { type: 'module' }
);
worker.postMessage(arrayBuffer, [arrayBuffer]);  // Transfer, don't copy
```

### State Management Patterns

**UI state** (Zustand) — for things that don't need server round-trips:

```tsx
import { useSettingsStore } from '../stores/settingsStore';

// Read state
const pointSize = useSettingsStore((s) => s.pointSize);

// Update state
useSettingsStore.getState().setPointSize(5);
```

**Server state** (React Query) — for API data:

```tsx
import { useQuery } from '@tanstack/react-query';

const { data, isLoading } = useQuery({
  queryKey: ['markers', datasetId],
  queryFn: () => apiFetch(`/datasets/${datasetId}/markers?format=json`),
  staleTime: 5 * 60 * 1000,  // Cache for 5 minutes
});
```

### Styling Conventions

- **Tailwind CSS v4** with utility classes (no CSS modules or styled-components)
- **Color scheme**: dark sidebar (`bg-slate-900`), light content area, teal accent (`text-teal-600`)
- **Spacing**: `p-6` for panel padding, `gap-4` for flex/grid gaps, `mb-4` for section spacing
- **Typography**: Inter font, `text-sm` for most content, `text-xs` for secondary labels
- **Responsive**: panels use flex layout that adapts, but primary target is desktop (1280px+)

## Common Patterns

### Error Handling in Endpoints

```python
@router.get("/datasets/{dataset_id}/something")
async def get_something(dataset_id: str, dm: DatasetManager = Depends(get_dataset_manager)):
    adaptor = await dm.get_or_load_dataset(dataset_id)
    if adaptor is None:
        raise HTTPException(status_code=404, detail="Dataset not found.")

    try:
        result = adaptor.some_method()
    except KeyError as e:
        raise HTTPException(status_code=404, detail=str(e))

    return result
```

### Frontend Error Boundaries

Each panel should be wrapped in an ErrorBoundary so one panel crash doesn't break the whole app:

```tsx
<ErrorBoundary fallback={<div>Something went wrong</div>}>
  <MyPanel />
</ErrorBoundary>
```

### Adding a New Preprocessing Detection Step

Edit `backend/src/scview/core/assessor.py`:

```python
def _check_my_step(adata) -> StepStatus:
    """Detect whether my preprocessing step has been done."""
    if "my_key" in adata.obs.columns:
        return StepStatus(done=True, confidence="high", details="my_key found in obs")
    return StepStatus(done=False, confidence="high", details="my_key not found in obs")
```

Then add it to the `PreprocessingState` model and `assess_preprocessing()` function.

### Adding a New Pipeline Step

Edit `backend/src/scview/core/pipeline.py`:

```python
def _run_my_step(adata, params: PipelineParams):
    """Execute my preprocessing step."""
    sc.some_function(adata, param=params.my_param)
```

Add the step name to the dependency ordering and the params to `PipelineParams`.

## Debugging Tips

- **Backend**: FastAPI auto-generates Swagger docs at http://localhost:8000/docs — use it to test endpoints interactively
- **Frontend**: React Query DevTools (if installed) show cached queries and their states
- **Arrow data**: Use `console.log(table.schema.toString())` to inspect Arrow table schemas in the browser
- **deck.gl**: Add `onError` and `onWarning` callbacks to DeckGL component for WebGL debugging
- **R converter**: Check `docker compose logs converter` for R error messages; the converter logs full stack traces
